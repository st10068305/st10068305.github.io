Using

Please pay careful attention to using the project demonstrated below.

In this tutorial, you will learn how to:

  • Use the project.

Before you start

Make sure that:

  • You have dotnet cli installed globally on your system.

  • You have cloned the project from here

  • or, downloaded and extracted a zip from the link above.

  • You have compiled the project.

  • You have run the project.

Let's Use It!

  1. Authentication Window

Welcome to the Authentication Window. Here you can register or login with a user to access the application.

  • Please make sure you have a good password so that no one else can interfere with your data.

  1. Semester Settings

By default, you will be greeted by the screen below to set up your new user.

  • Please be careful with what date and number of weeks you decide to use.

  1. Main Window

After you have set up your new user, you will be greeted by the screen below.

  • On this window you can

    • Add modules

    • Add hours for modules

    • Delete modules

    • Update your semester settings

    • Logout

Hover over buttons to see what they do.

  1. Adding A Module

When clicking "Add Module" you will see the below dialog open.

  • Here you can fill in the modules details and add it.

  1. Updating Semester Settings

When clicking "Update Semester Settings" you will see the Semester Settings again.

  • You can modify your semester settings, but remember to be careful what date and number of weeks you choose.

  1. Adding Hours For A Module

When clicking "Record Self Study Hours" you will see the below dialog open.

  • Here you can choose a date to record the hour/s on.

  1. Deleting A Module

When clicking "Delete Module" you will see a confirmation dialog as below open.

  • Be wise, once you delete something, it is gone forever.

  1. Logout

When clicking "Logout" you will see the authentication window and will be logged out of the main application.
