Running

Please pay careful attention to running the project demonstrated below.

In this tutorial, you will learn how to:

  • Run the project using Visual Studio

  • Run the project using Jetbrains Rider

Before you start

Make sure that:

  • You have dotnet cli installed globally on your system.

  • You have cloned the project from here

  • or, downloaded and extracted a zip from the link above.

  • You have compiled the project.

Lets Run It!

Open a terminal window in the folder of your cloned/extracted version of the github project.

dotnet run
